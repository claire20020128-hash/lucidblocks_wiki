## 使用说明

1. **按顺序检查** - 从第1项到第16项依次检查
2. **逐项验证** - 每一项都要验证所有子项
3. **记录问题** - 如果某项验证失败，记录下来
4. **修复问题** - 修复所有验证失败的项
5. **重新验证** - 修复后重新验证该项
6. **最终检查** - 所有项都通过后，进行总体验证

## 验证命令参考

```bash
# 检查是否有旧项目名称
grep -r "旧项目名" src/app/[locale]/page.tsx src/locales/en.json

# 检查是否有内部链接
grep -E 'href="/' src/app/[locale]/page.tsx

# 检查是否有emoji
grep -E '😀|😁|😂|🎉' src/app/[locale]/page.tsx src/locales/en.json

# 检查是否有硬编码颜色
grep -E '#[0-9a-fA-F]{6}|rgb\(' src/app/[locale]/page.tsx

# 检查TypeScript语法
npx tsc --noEmit src/app/[locale]/page.tsx

# 检查JSON格式
python3 -m json.tool src/locales/en.json > /dev/null
```
