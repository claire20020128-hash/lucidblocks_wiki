先把你要的“这个东西”说清楚：你问的 **“Heartopia 用的粉色系具体色值”**，如果官方公开了品牌手册（Brand Guideline）就能直接给到“官方指定 HEX”。但目前能稳定拿到的“官方素材”主要是 **Steam 商店页的头图/宣传截图**，这些属于官方发布的视觉资产，我下面的色值是**从这些官方宣传图里抽样提取出来的主色**（更接近“实际视觉用色”，但不等同于“官方手册指定色值”）。 ([Steam 商店][1])

## 结论：Heartopia 主粉色（建议选「1 浅粉色主题」+ 粉紫点缀）

从官方宣传图里出现频率最高、最“Heartopia”的粉系，我建议你用这 **3 个做主粉**（你要 2-3 个，我给 3 个，方便做层级）：

* **主粉紫（Brand Lilac Pink）**：`#B77ED7`（偏粉紫的主识别色，适合按钮描边/标题强调）
* **浅棉花糖粉（Soft Pink）**：`#F7B8F9`（适合背景高光、徽标底、hover 光晕）
* **柔雾玫瑰粉（Blush Rose）**：`#E7AAD8`（适合标签、插画边缘、二级强调）

> 这套更贴近 Heartopia 的“软糯、治愈、轻社交”气质，而不是偏赛博的深粉紫。

## 完整色彩方案（可直接上站）

下面给你一套 **Light 主题（推荐）** + **可选 Dark 主题**，并把“怎么用”也写清楚。

### 1) Light Theme（治愈浅粉主调）

**中性色（页面底/卡片底）**（从官方 UI 面板观感抽样，偏奶油白）：

* Page Background：`#FFF9FD`（全站底色，几乎白但带一点粉气）
* Surface / Card：`#FAF7F2`（卡片底，温柔不刺眼）
* Surface-2：`#EEEBE4`（二层容器/表格头）
* Border：`#E9D8F3`（浅紫边框，和主粉紫同系）

**品牌粉（你要的重点）**

* Pink-Primary：`#B77ED7`
* Pink-Soft：`#F7B8F9`
* Pink-Blush：`#E7AAD8`

**可用的“官方气质”辅助色（从宣传图里很常见）**

* Sky Blue（清新天空蓝）：`#88C6F3`（链接/信息提示/插画辅助）
* Mint Teal（薄荷青）：`#37CDC7`（次按钮/成功态/小功能按钮）
* Sunshine Orange（暖橙点睛）：`#FD9A35`（警告/重要提示/“新内容”徽标）

**文字色（保证可读性）**

* Text Primary：`#1F2430`
* Text Secondary：`#4B5563`
* Text Muted：`#6B7280`

**语义色（做状态用，不要硬用大红大绿）**

* Success：`#1F7A77`（配白字更稳）
* Warning：`#B15A02`（配白字可读）
* Error：`#B02A5B`（莓果红，和粉系协调）

---

### 2) Primary 按钮怎么配色（避免“粉底+白字看不清”）

你如果直接用 `#B77ED7` 做按钮底色，**白字对比度不够**（会发灰）。更稳的做法：

* **主按钮（CTA）底色用更深一档**：`#9D50C9`（白字可读）
* Hover：`#8D3ABB`
* Active：`#7B33A4`

这样既保留 Heartopia 的粉紫气质，又不会“可爱但看不清”。

## 可直接复制的 CSS Tokens（Light + Dark）

```css
:root {
  /* Neutrals */
  --bg: #FFF9FD;
  --surface: #FAF7F2;
  --surface-2: #EEEBE4;
  --border: #E9D8F3;

  /* Text */
  --text: #1F2430;
  --text-2: #4B5563;
  --text-3: #6B7280;

  /* Brand Pinks (Heartopia vibe) */
  --pink-primary: #B77ED7;
  --pink-soft: #F7B8F9;
  --pink-blush: #E7AAD8;

  /* CTA (accessible) */
  --primary: #9D50C9;
  --primary-hover: #8D3ABB;
  --primary-active: #7B33A4;
  --on-primary: #FFFFFF;

  /* Supporting accents */
  --sky: #88C6F3;
  --teal: #37CDC7;
  --orange: #FD9A35;

  /* Semantic */
  --success: #1F7A77;
  --warning: #B15A02;
  --error: #B02A5B;

  /* Effects */
  --focus: rgba(247, 184, 249, 0.55); /* from --pink-soft */
  --shadow: 0 10px 30px rgba(157, 80, 201, 0.18);
}

/* Optional dark mode */
[data-theme="dark"] {
  --bg: #120A18;
  --surface: #1A1022;
  --surface-2: #22122E;
  --border: rgba(233, 216, 243, 0.18);

  --text: #F6F2FA;
  --text-2: #D7CFE2;
  --text-3: #B8AEC7;

  --primary: #B77ED7;
  --primary-hover: #C99EE1;
  --primary-active: #DEC5ED;
  --on-primary: #1F2430;

  --shadow: 0 12px 40px rgba(0, 0, 0, 0.35);
}
```

## 通俗示例：首页怎么用这套色

* **Hero 区大按钮（Download / Get Codes）**：背景用 `--primary`，文字 `--on-primary`，Hover 用 `--primary-hover`。
* **功能卡片（Tools / Guides）**：卡片底 `--surface`，边框 `--border`，标题 `--text`，小标签底用 `--pink-soft`（文字用 `--text`）。
* **“New / Updated”徽标**：底色用 `--orange`，文字用白色或深色都行（更推荐深底 `--warning` + 白字）。

## 你想自己“验色/扩展色阶”的最省事方法

用【Adobe Color】或【Coolors】的“Extract Palette”：
1）打开 Heartopia 的 Steam 宣传图（就是我引用的那些图）
2）把图片丢进【Adobe Color】→ Extract Theme
3）把你喜欢的 2-3 个粉色钉住，再让工具自动补齐互补色/中性色
这样你就能做出“更像官方但更适合你站点 UI 的一整套”。
