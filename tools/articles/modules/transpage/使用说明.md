## 🚀 快速开始

### 执行翻译

```bash
# 进入脚本目录
cd tools/articles/modules/transpage

# 翻译所有配置的语言（推荐）
python3 translate-messages-enhanced.py --overwrite --report

# 翻译指定语言
python3 translate-messages-enhanced.py --lang pt,fr,es --overwrite --report

# 增量翻译（只翻译变化的部分）
python3 translate-messages-enhanced.py --incremental --report

# 指定分块策略（如果默认失败）
python3 translate-messages-enhanced.py --lang pt --strategy small --overwrite --report
```

### 验证结果

```bash
# 1. JSON 格式验证
for lang in pt fr es de it ja ko; do
  python3 -m json.tool src/locales/$lang.json > /dev/null && echo "✅ $lang" || echo "❌ $lang"
done

# 2. 查看翻译报告
cat temp/translation-reports/*.json

# 3. 检查字段数量
en_fields=$(grep -o '"[^"]*":' src/locales/en.json | wc -l)
for lang in pt fr es de it ja ko; do
  lang_fields=$(grep -o '"[^"]*":' src/locales/$lang.json | wc -l)
  echo "$lang: $lang_fields 字段 (en: $en_fields)"
done

# 4. 本地测试
npm run dev
# 在浏览器中切换语言检查
```

## ⚙️ 配置参数说明

### 必须修改的参数

| 参数 | 说明 | 示例 |
|------|------|------|
| `api_key` | API 密钥 | `"sk-xxx..."` |
| `languages` | 目标语言列表 | `["pt", "fr", "es"]` |
| `output_dir` | 输出目录 | `"src/locales/"` |
| `protected_terms.game_names` | 游戏名称（不翻译） | `["Your Game"]` |
| `game_names` | 每种语言的游戏名 | `{"en": "Your Game"}` |

### 推荐保持的参数（已优化）

| 参数 | 推荐值 | 说明 |
|------|--------|------|
| `temperature` | `0.1` | 最低随机性，防止字段错位 |
| `use_value_extraction` | `false` | 完整JSON模式，防止错位 |
| `retry_attempts` | `8` | 增加重试次数 |
| `retry_delay` | `10` | 重试延迟（秒） |
| `max_tokens` | `32768` | 足够大，避免截断 |
| `timeout` | `1800` | 30分钟超时 |

### 可选参数

| 参数 | 说明 | 默认值 |
|------|------|--------|
| `model` | AI 模型 | `"gemini-2.0-flash"` |
| `protected_terms.character_names` | 角色名（不翻译） | `[]` |
| `protected_terms.technical_terms` | 技术术语（不翻译） | `["PvP", "PvE"]` |

## 🔍 命令行参数

```bash
python3 translate-messages-enhanced.py [选项]

选项：
  --lang LANG           目标语言（逗号分隔），如：pt,fr,es
  --overwrite           覆盖已有翻译
  --incremental         增量翻译（只翻译变化的部分）
  --strategy STRATEGY   分块策略：top_level, medium, small, tiny
  --report              生成详细报告
```

### 使用示例

```bash
# 翻译所有配置的语言
python3 translate-messages-enhanced.py --overwrite --report

# 只翻译葡萄牙语和法语
python3 translate-messages-enhanced.py --lang pt,fr --overwrite

# 增量翻译（en.json 有小改动时）
python3 translate-messages-enhanced.py --incremental --report

# 使用小分块策略（默认失败时）
python3 translate-messages-enhanced.py --lang pt --strategy small --overwrite

# 使用最小分块（最后手段）
python3 translate-messages-enhanced.py --lang pt --strategy tiny --overwrite
```

## 📊 分块策略说明

| 策略 | 分块大小 | 适用场景 |
|------|---------|---------|
| `top_level` | 按顶层键分块 | 默认，适合大多数情况 |
| `medium` | 30个键一块 | 顶层键过大时 |
| `small` | 15个键一块 | 中等分块失败时 |
| `tiny` | 5个键一块 | 最后手段，最慢但最稳定 |

脚本会自动降级重试：`top_level` → `medium` → `small` → `tiny`

## 🐛 常见问题

### Q1: API 连接超时
```bash
# 解决方案 1: 增加超时时间
# 编辑 transpage_config.json: "timeout": 3600

# 解决方案 2: 分批翻译
python3 translate-messages-enhanced.py --lang pt --overwrite
python3 translate-messages-enhanced.py --lang fr --overwrite
python3 translate-messages-enhanced.py --lang es --overwrite
```

### Q2: 字段错位
```bash
# 确认配置
grep "use_value_extraction" transpage_config.json
# 应该显示: "use_value_extraction": false

grep "temperature" transpage_config.json
# 应该显示: "temperature": 0.1

# 如果不对，修改配置后重新翻译
```

### Q3: 部分模块未翻译
```bash
# 查看报告
cat temp/translation-reports/*.json

# 使用更小的分块重新翻译
python3 translate-messages-enhanced.py --lang pt --strategy small --overwrite
```

### Q4: 游戏名被翻译了
```bash
# 检查配置
grep -A 5 "protected_terms" transpage_config.json

# 确保游戏名在 protected_terms.game_names 中
# 修改后重新翻译
```

### Q5: JSON 格式错误
```bash
# 验证 JSON
python3 -m json.tool src/locales/pt.json

# 如果有错误，查看具体位置并手动修复
# 或者重新翻译
python3 translate-messages-enhanced.py --lang pt --overwrite
```
