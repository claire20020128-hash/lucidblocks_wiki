## 翻译首页多语言文件 - 快速参考

### 📝 执行命令（推荐使用增强版）

```bash
cd tools/articles/modules/transpage
python3 translate-messages-enhanced.py --overwrite --report
```

### ⚙️ 配置检查（首次使用必做）

编辑 `transpage_config.json`，确认以下配置：

```json
{
  "temperature": 0.1,              // ✅ 必须 ≤ 0.3
  "use_value_extraction": false,   // ✅ 必须 false（防止字段错位）
  "retry_attempts": 8,              // ✅ 建议 ≥ 8
  "retry_delay": 10,                // ✅ 建议 ≥ 10

  "languages": ["pt", "fr", "es", "de", "it", "ja", "ko"],  // 根据需求修改

  "protected_terms": {
    "game_names": ["Anime Tactical Simulator"],  // 改为你的游戏名
    "character_names": [],                        // 添加角色名
    "technical_terms": ["PvP", "PvE", "HP"]      // 添加术语
  }
}
```

### ✅ 翻译后验证（必做）

```bash
# 1. JSON 格式验证
for lang in pt fr es de it ja ko; do
  python3 -m json.tool src/locales/$lang.json > /dev/null && echo "✅ $lang" || echo "❌ $lang"
done

# 2. 查看报告
cat temp/translation-reports/*.json

# 3. 本地测试
npm run dev
```

### 🔍 常见问题快速修复

| 问题 | 快速解决 |
|------|---------|
| API 超时 | 分批翻译：`--lang pt` → `--lang fr` |
| 字段错位 | 确认 `use_value_extraction: false` |
| 部分未翻译 | 使用小分块：`--strategy small` |
| 游戏名被翻译 | 检查 `protected_terms.game_names` |
| JSON 格式错误 | 重新翻译：`--lang pt --overwrite` |
