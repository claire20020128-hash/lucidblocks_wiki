## 🔧 三个问题的修复代码

### 1. 路径解析修复

```python
def find_project_root():
    """查找项目根目录（包含 package.json）"""
    current = Path(script_dir)
    for _ in range(5):
        if (current / 'package.json').exists():
            return current
        current = current.parent
    return current

project_root = find_project_root()
locales_dir = project_root / 'src' / 'locales'
```

### 2. API 响应修复

```python
# 支持流式和非流式响应
async def parse_api_response(response, target_lang):
    content_type = response.headers.get('Content-Type', '')
    if 'text/event-stream' in content_type:
        return await parse_stream_response(response, target_lang)
    else:
        return await parse_json_response(response, target_lang)
```

### 3. Windows 编码修复

```python
# -*- coding: utf-8 -*-
import sys, os, io

if sys.platform == 'win32':
    sys.stdout = io.TextIOWrapper(sys.stdout.buffer, encoding='utf-8', errors='replace')
    sys.stderr = io.TextIOWrapper(sys.stderr.buffer, encoding='utf-8', errors='replace')
    os.system('chcp 65001 > nul 2>&1')
```
