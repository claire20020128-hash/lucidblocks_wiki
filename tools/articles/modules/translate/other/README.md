# 旧代码备份

这个目录包含被废弃的多语言翻译实现。

## 文件说明

- `old_translator.py` - 旧的翻译器实现（多语言单次调用）
- `old_translation-prompt-template.md` - 旧的多语言翻译prompt模板

## 废弃原因

1. **性能问题**: 多语言单次调用速度太慢
   - 单次API需要生成3种语言的完整内容
   - 输出tokens数量 × 3，导致生成时间过长
   - max_tokens=12288，每次都要生成接近上限

2. **可靠性问题**: 一次失败影响所有语言
   - 如果API调用失败，3种语言都无法生成
   - 重试成本高

3. **并发限制**: 无法充分利用API并发能力
   - 并发度只有文章数N
   - 无法达到最大并发（文章数 × 语言数）

## 新实现

- **位置**: `tools/articles/modules/translate/`
- **策略**: 单语言翻译 + 高并发
- **性能提升**: 60-70%
- **并发度**: 文章数 × 语言数
