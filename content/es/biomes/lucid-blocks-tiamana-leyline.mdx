<Section icon={Map} title="Dominando el Equipo de Traslado y Movimiento">

Caminar es ineficiente en los paisajes verticales de Lucid Blocks. Para desbloquear los logros de traslado, debes fabricar equipo especializado que permita el vuelo y el agarre rápido. Dominar la combinación del Guante Planeador y el Gancho es la única forma de alcanzar las ubicaciones más aisladas de la **leyline tiamana de lucid blocks** repartidas por el mapa.

| Objeto | Receta | Logro | Caso de Uso |
| :--- | :--- | :--- | :--- |
| **Glider Glove** | Lana + Pluma (Hueso + Lana) | **Virtue α** | Planea desde picos altos para cubrir distancia |
| **Hookshot** | Bloque de Metal + Cuerda | **Virtue β** | Engánchate a superficies y mantén el impulso |
| **Feather** | Hueso + Lana | N/A | Componente para el Guante Planeador |

<Callout type="info" title="Combo de Movimiento">
Puedes combinar el impulso del Gancho con el Guante Planeador. Lánzate a través de brechas masivas enganchándote a un punto alto y desplegando el planeador en el punto máximo de tu balanceo.
</Callout>

</Section>

<Section icon={Swords} title="El Arsenal Mágico: Artesanía de Varitas">

El combate en Lucid Blocks gira en torno a los logros "Archangel", que requieren que empuñes armas mágicas específicas. Aunque ocasionalmente puedes encontrarlas en enemigos como los Manikins, fabricarlas asegura que tengan la durabilidad necesaria para las peleas contra jefes.

| Tipo de Varita | Cadena de Recetas | Logro |
| :--- | :--- | :--- |
| **Ball Wand** | Magnetita + Palo = Hacha Llave; Hacha Llave + Magnetita | **Archangel α** |
| **Faith Wand** | Varita de Burbujas + Hueso = Varita Amiga; Varita Amiga + Hueso | **Archangel β** |
| **Explosive** | Bloque Accelo + Esencia de Bola (o Fuego + Gancho + Varita de Burbujas) | **Archangel γ** |

<Callout type="warning" title="Durabilidad de Armas">
Las varitas soltadas por enemigos suelen tener una durabilidad muy baja. Siempre es más seguro fabricar la tuya desde cero para asegurar que no se rompa durante un encuentro crítico.
</Callout>

</Section>

<Section icon={Target} title="Manipulación del Ecosistema y Farmeo">

Para desbloquear los logros "Dominion" y "Apotheosis β", debes jugar a ser Dios con la flora y fauna local. Esto implica una cadena específica de fusiones para crear formas de vida que proporcionan recursos infinitos.

<Steps>
  <Step num="1" title="Crear Bloques de Hierba">
    Mezcla **Tierra** y **Arena** para crear un Bloque de Hierba. Este sirve como base para la mayoría de las fusiones orgánicas.
  </Step>
  <Step num="2" title="Recolectar Partes de Criaturas">
    Mata a una criatura local para recolectar un segmento o glándula. Combina esto con el **Bloque de Hierba** para generar un **Huevo de Bubblebear**.
  </Step>
  <Step num="3" title="Generar la Fruit Femme">
    Fusiona el **Huevo de Bubblebear** con otro **Bloque de Hierba** para crear una **Cápsula de Fruit Femme**. Esta criatura proporciona comida infinita una vez que crece.
  </Step>
  <Step num="4" title="Fertilización y Cosecha">
    Mata a la Fruit Femme para obtener semillas. Plántalas, usa fertilizante para desbloquear **Dominion α**, y espera a que crezca una perla para desbloquear **Dominion β**.
  </Step>
</Steps>

<Callout type="success" title="Bucle de Comida Infinita">
La Cápsula de Fruit Femme es uno de los objetos más valiosos del juego. Manténla a salvo en tu Qualia para asegurar que nunca te quedes sin objetos para restaurar el hambre.
</Callout>

</Section>

<Section icon={Shield} title="Leylines Tiamana y el Jefe Final">

El juego final comienza cuando empiezas a cazar la **leyline tiamana de lucid blocks**. Estos bloques de teletransporte blanco brillante son la única forma de completar tu Rosario y convocar al jefe final. Encontrar cada **leyline tiamana de lucid blocks** requiere un nivel de experiencia significativo y las mejoras adecuadas.

| Logro | Requisito | Estrategia |
| :--- | :--- | :--- |
| **Angel γ** | Descubrir una leyline Tiamana | Busca bloques blancos brillantes en la naturaleza |
| **Angel δ** | Superar un desafío Tiamana | Entra en el cubo blanco y derrota la prueba en su interior |
| **Apotheosis γ** | Fusionar un artefacto divino | Mezcla Bloque de Hierba + Esencia de Bola (de espíritus Kodama) |
| **Angel ε** | Llegar al jefe final | Completa el Rosario despejando todas las leylines |

<Callout type="info" title="Rastreo de Leylines">
La forma más rápida de encontrar una **leyline tiamana de lucid blocks** es elegir la **Mejora de Colgante** durante una subida de nivel. Esta mejora revela las ubicaciones de las leylines cercanas en tu mapa.
</Callout>

Para farmear la experiencia necesaria para estas mejoras, usa el **Bucle de Fusión Infinita**:
1. Fusiona una **Cápsula de Goo** con cualquier objeto aleatorio para obtener una **Cápsula de Gel**.
2. Fusiona la **Cápsula de Gel** con un objeto para obtener **dos Cápsulas de Goo**.
3. Repite este proceso para generar criaturas infinitas y obtener grandes ganancias de XP.

</Section>
