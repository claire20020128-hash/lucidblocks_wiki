**Lucid Blocks 전체 화면**을 활성화하려면 게임 플레이 중 언제든지 **F11**을 누르기만 하면 됩니다. 셰이더 효과와 특정 생태계에서 사용된 "프랙탈" 수학을 온전히 감상하기 위해 이 설정을 강력히 권장합니다.

| 동작 | 키 / 입력 | 설명 |
| :--- | :--- | :--- |
| **전체 화면** | **F11** | 창 모드와 전체 화면 모드 사이를 전환 |
| **이동** | **WASD** | 표준 방향 이동 |
| **점프** | **Space** | 수직 이동 및 등반 |
| **앉기** | **Left Ctrl** | 가장자리에서 떨어지는 것을 방지 (디바이드에서 필수) |
| **인벤토리** | **E** | 메인 메뉴 및 아포테오시스 탭 열기 |
| **블록 파괴** | **왼쪽 클릭** | 표준 상호작용/공격 |
| **블록 배치** | **오른쪽 클릭** | 보조 상호작용/건설 |
| **아이템 버리기** | **Q** | 활성 슬롯에서 아이템 버리기 |

<Callout type="tip" title="프로 팁: 앉기의 안전성">
지표면과 하부 세계를 나누는 두꺼운 천장인 "디바이드(The Divide)"를 탐험할 때는 항상 **Left Ctrl**을 누르고 계세요. 갈고리나 글라이더를 장착하지 않은 상태에서 디바이드에서 떨어지는 것은 거의 항상 치명적입니다.
</Callout>

<Section icon={Sparkles} title="아포테오시스 연금술 시스템">

Lucid Blocks에서의 제작은 **아포테오시스(Apotheosis)**라고 불립니다. 이것은 전통적인 조립보다는 변형이나 연금술에 가깝게 작동합니다. 물질은 한 상태에서 다른 상태로 전이되며, 종종 예측 불가능하고 초현실적인 결과를 초래합니다. 단순히 도구를 "만드는" 것이 아니라, 수집한 재료로부터 도구를 "환기"시키는 것입니다.

<Steps>
  <Step num="1" title="아포테오시스 탭 접속">
    **E**를 눌러 인벤토리를 열고 화면상의 버튼이나 키보드 화살표를 사용하여 **아포테오시스(Apotheosis)** 탭으로 이동합니다.
  </Step>
  <Step num="2" title="재료 선택">
    원형 연금술 인터페이스 내의 슬롯에 두 가지 다른 재료를 배치합니다. 특정 슬롯은 중요하지 않으며, 아이템의 유형과 양만이 중요합니다.
  </Step>
  <Step num="3" title="경험치 획득">
    모든 성공적인 변환은 보이지 않는 XP를 부여합니다. 게이지가 가득 차면 "블룸(Bloom)" 상태가 되어 능력치를 업그레이드할 수 있습니다.
  </Step>
  <Step num="4" title="결과물 회수">
    새로운 아이템이 중앙에 나타납니다. 일부 레시피는 호환 가능합니다. 예를 들어, 갈고리는 마나 젤(Mana Gel)이나 수은 덩굴(Mercury Tendrils)로 만들 수 있습니다.
  </Step>
</Steps>

| 결과 아이템 | 재료 A | 재료 B | 효과 / 용도 |
| :--- | :--- | :--- | :--- |
| **나무 곡괭이** | 나무 블록 | 막대기 | 기본 채광 도구 |
| **갈고리** | 마나 젤 | 실 | 수직 이동에 필수적 |
| **오리 팔찌** | 모래 | 양모 | 공기 +5, HP +2 (장착 가능) |
| **이름 짓기 연필** | 책 | 실 | 아이템이나 엔티티의 이름을 짓는 데 사용 |
| **물 양동이** | 빈 양동이 | 나무 블록 | 나무를 액체 상태의 물로 변환 |
| **노(Paddle)** | 곡괭이 | 생 양고기 | 이상하고 "찌르는 듯한" 근접 무기 |

<Callout type="warning" title="사망 시 인벤토리 변환">
Lucid Blocks에서 죽음은 변형적인 사건입니다. 인벤토리에 있는 모든 것이 무작위로 함께 "아포테오시스(변환)"됩니다. 원래 아이템은 잃게 되지만, 이전 장비로 제작된 완전히 새롭고 희귀한 재료를 가지고 부활할 수도 있습니다.

<Section icon={Swords} title="엔티티: 아군과 적군">

Lucid Blocks의 엔티티는 평범한 것부터 수학적으로 복잡한 것까지 다양합니다. 많은 엔티티가 이동을 위해 인버스 키네마틱스(Inverse Kinematics)를 사용하여, 꿈의 미학에 걸맞은 불안정하고 부자연스러운 모습을 보여줍니다.

| 엔티티 이름 | 행동 | 드롭 아이템 | 적대성 |
| :--- | :--- | :--- | :--- |
| **마네킹** | 아이템(검/지팡이)을 들고 있음 | 실, 나무 판자 | 중립 (공격 시 적대적) |
| **블래스퍼미(Blasphemy)** | 큐브 머리를 가진 거미 형상 | 킬라(Kila, 희귀 아이템) | **적대적** |
| **테토 페어(Tetto Pair)** | "보컬로이드" 소리를 냄 | 알 수 없음 | 우호적/초현실적 |
| **블록 스파이더** | 일반 블록으로 위장 | 거미줄 | **적대적** |
| **규조류(Diatom)** | 사람 얼굴이 있는 구체 | 없음 | **적대적** (느리게 이동) |
| **양** | 탑승 가능 | 양모, 양고기 | 우호적 |

<Callout type="success" title="탑승 메커니즘">
양 위에 올라타서 실제로 양을 탈 수 있습니다. 직접 조종할 수는 없지만, 양을 때리면 무작위 방향으로 달려가므로 혼란스럽지만 빠른 지표면 이동 수단이 됩니다.
</Callout>
