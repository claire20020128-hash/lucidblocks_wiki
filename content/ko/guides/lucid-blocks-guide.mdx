### 에센스 로직의 이해
게임 내 모든 아이템은 숨겨진 에센스 값을 가지고 있습니다. 아이템을 조합하면 게임은 지배적인 에센스를 계산하여 결과물을 결정합니다. 상충하는 에센스를 섞으면 종종 '쓰레기'나 의도치 않은 부작용이 있는 기괴한 유물이 만들어집니다.

| 에센스 유형 | 일반적인 출처 | 결과 카테고리 |
| :--- | :--- | :--- |
| **유기물** | 막대기, 잎사귀, 양털 | 기본 도구, 유틸리티 장비 |
| **광물** | 자갈 블록, 자철석 | 중화기, 채광 도구 |
| **에테르** | 선(Glands), 수은 촉수 | 마법 지팡이, 유물 |
| **휘발성** | 유황, 글래글 토큰 | 폭발물, 화염 무기 |

</Section>

<Section icon={Target} title="필수 초반 게임 제작 레시피">

Lucid Blocks에서의 첫 한 시간은 신뢰할 수 있는 수확 도구 세트를 확보하는 데 투자해야 합니다. 이 도구들이 없으면 고티어 융합에 필요한 재료를 모을 수 없습니다. 다음 표는 모든 플레이어가 외워야 할 가장 확실한 스타터 레시피를 요약한 것입니다.

| 아이템 이름 | 재료 | 주요 용도 |
| :--- | :--- | :--- |
| **나무 곡괭이** | 막대기 + (흙 / 통나무 / 모래) | 기본 블록 수확 |
| **잎사귀 곡괭이** | 막대기 + 나무 곡괭이 | 빠른 잎/열매 수확 |
| **뼈 곡괭이** | 뼈 + 실 | 내구성이 좋은 초반 채광 |
| **서리 곡괭이** | 막대기 + 오스뮴 | 추운 바이옴에서의 고속 채광 |
| **곡괭이** | 나무 곡괭이 + 알루미늄 (x3) | 표준 중간 티어 채광 도구 |
| **자두 도끼** | 나무 곡괭이 + 자철석 | 효율적인 나무 수확 |

<Callout type="warning" title="도구 내구도">
**잎사귀 곡괭이**와 같은 초반 도구는 매우 빨리 파손됩니다. 지속적인 수리에 자원을 낭비하지 않으려면 원래 목적(잎 수확)으로만 사용하세요.

### 마법 지팡이와 환경 조작
지팡이는 **볼 지팡이(Ball Wand)** 베이스에 다양한 생물학적 또는 광물 구성 요소를 결합하여 제작합니다. 이 아이템들은 단순히 데미지만 입히는 것이 아니라 주변 세계의 규칙을 바꿉니다.

| 지팡이 이름 | 레시피 | 효과 |
| :--- | :--- | :--- |
| **볼 지팡이** | 프렌드 지팡이 + 실 | 약하고 기본적인 투사체 발사 |
| **프렌드 지팡이** | 볼 지팡이 + 선(Gland) | 하급 아군 소환 또는 버프 |
| **헤이트 지팡이** | 볼 지팡이 + 생양고기 | 강력한 광역 데미지 및 블록 파괴 |
| **페이스 지팡이** | 프렌드 지팡이 + 뼈 곡괭이 | 예측 불가능한 환경 변화 |

<Section icon={Sparkles} title="이동을 위한 최고의 아이템 5선">

이동은 **Lucid Blocks 가이드**에서 가장 만족스러운 부분입니다. 특정 유물을 조합하면 맵 전체를 효율적으로 날아다닐 수 있으며, 낙하 데미지를 무효화하고 도달할 수 없는 높이에 도달할 수 있습니다.

**최고의 이동 아이템:**
- **벌 장갑(Bee Glove)**: 절대적인 필수 아이템. 활공이 가능하며 올바르게 사용하면 낙하 데미지를 무효화합니다.
- **그래플러(Grappler)**: 벌 장갑과 함께 사용하면 플레이어를 놀라운 속도로 발사합니다.
- **헤이즈 핸드(Haze Hand)**: 데미지는 전혀 없지만, 적(또는 자신)을 공중으로 격렬하게 날려 보낼 수 있습니다.
- **다람쥐 반지(Squirrel Ring)**: 점프 높이를 높여주며, 수직 플랫폼 이동에 필수적입니다.
- **황금 부츠(Golden Boots)**: 체력을 +4 증가시키지만 무게 물리학을 급격하게 변화시켜 독특한 이동 기술을 가능하게 합니다.
