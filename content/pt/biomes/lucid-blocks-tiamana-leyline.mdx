<Section icon={Target} title="Manipulação de Ecossistema e Cultivo">

Para desbloquear as conquistas "Dominion" e "Apotheosis β", você deve agir como um deus com a flora e fauna locais. Isso envolve uma cadeia específica de fusões para criar formas de vida que fornecem recursos infinitos.

<Steps>
  <Step num="1" title="Criar Blocos de Grama">
    Misture **Terra** e **Areia** para criar um Bloco de Grama. Isso serve como base para a maioria das fusões orgânicas.
  </Step>
  <Step num="2" title="Coletar Partes de Criaturas">
    Mate uma criatura local para coletar um segmento ou glândula. Combine isso com o **Bloco de Grama** para gerar um **Ovo de Bubblebear**.
  </Step>
  <Step num="3" title="Gerar a Fruit Femme">
    Funda o **Ovo de Bubblebear** com outro **Bloco de Grama** para criar uma **Cápsula de Fruit Femme**. Esta criatura fornece comida infinita uma vez crescida.
  </Step>
  <Step num="4" title="Fertilização e Colheita">
    Mate a Fruit Femme para obter sementes. Plante-as, use fertilizante para desbloquear **Dominion α**, e espere que ela cresça uma pérola para desbloquear **Dominion β**.
  </Step>
</Steps>

<Callout type="success" title="Loop de Comida Infinita">
A Cápsula de Fruit Femme é um dos itens mais valiosos do jogo. Mantenha-a segura em sua Qualia para garantir que você nunca fique sem itens de restauração de fome.
</Callout>

</Section>

<Section icon={Shield} title="Tiamana Leylines e o Chefe Final">

O final do jogo começa quando você inicia a caçada pela **lucid blocks tiamana leyline**. Esses blocos de teletransporte brancos brilhantes são a única maneira de completar seu Rosário e invocar o chefe final. Encontrar cada **lucid blocks tiamana leyline** requer um nível significativo e as melhorias certas.

| Conquista | Requisito | Estratégia |
| :--- | :--- | :--- |
| **Angel γ** | Descobrir uma Tiamana leyline | Procure por blocos brancos brilhantes na natureza |
| **Angel δ** | Superar um desafio Tiamana | Entre no cubo branco e vença a provação interna |
| **Apotheosis γ** | Fundir um artefato divino | Misture Bloco de Grama + Essência de Bola (de espíritos Kodama) |
| **Angel ε** | Alcançar o chefe final | Complete o Rosário limpando todas as leylines |

<Callout type="info" title="Rastreamento de Leyline">
A maneira mais rápida de encontrar uma **lucid blocks tiamana leyline** é escolher a **Melhoria de Pingente** durante um aumento de nível. Esta melhoria revela as localizações das leylines próximas no seu mapa.
</Callout>

Para farmar a experiência necessária para essas melhorias, use o **Loop de Fusão Infinita**:
1. Funda uma **Cápsula de Goo** com qualquer item aleatório para obter uma **Cápsula de Gel**.
2. Funda a **Cápsula de Gel** com um item para obter **duas Cápsulas de Goo**.
3. Repita este processo para gerar criaturas infinitas para ganhos massivos de XP.

</Section>
