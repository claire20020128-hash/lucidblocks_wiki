### エッセンスの論理を理解する
ゲーム内のすべてのアイテムには、隠されたエッセンスの値があります。アイテムを組み合わせると、ゲームは支配的なエッセンスを計算して出力を決定します。相反するエッセンスを混ぜると、多くの場合「ジャンク」や、意図しない副作用を持つ奇妙なアーティファクトが出来上がります。

| エッセンスタイプ | 主なソース | 結果として得られるカテゴリ |
| :--- | :--- | :--- |
| **有機物 (Organic)** | 棒、葉、羊毛 | 基本ツール、ユーティリティ装備 |
| **鉱物 (Mineral)** | 小石ブロック、磁鉄鉱 | 重武器、採掘ツール |
| **エーテル (Ethereal)** | 分泌腺、水銀の触手 | 魔法の杖、アーティファクト |
| **揮発性 (Volatile)** | 硫黄、グラグルトークン | 爆発物、火属性武器 |

</Section>

<Section icon={Target} title="序盤の必須クラフトレシピ">

Lucid Blocks での最初の 1 時間は、信頼できる採取ツールのセットを確保することに費やすべきです。これらがなければ、上位ティアの融合に必要な材料を集めることができません。以下の表は、すべてのプレイヤーが覚えておくべき、最も信頼性の高いスターターレシピをまとめたものです。

| アイテム名 | 材料 | 主な用途 |
| :--- | :--- | :--- |
| **木製のつるはし** | 棒 + (土 / 丸太 / 砂) | 基本的なブロック採取 |
| **葉のつるはし** | 棒 + 木製のつるはし | 葉やベリーの高速採取 |
| **骨のつるはし** | 骨 + 糸 | 耐久性のある序盤の採掘 |
| **霜のつるはし** | 棒 + オスミウム | 寒冷バイオームでの高速採掘 |
| **ピッケル** | 木製のつるはし + アルミニウム (x3) | 標準的な中盤採掘ツール |
| **プラムの斧** | 木製のつるはし + 磁鉄鉱 | 効率的な木材採取 |

<Callout type="warning" title="ツールの耐久度">
**葉のつるはし**のような序盤のツールは非常に早く壊れます。絶え間ない修理でリソースを無駄にしないよう、本来の目的（葉の採取）にのみ使用してください。
</Callout>

### ツール進行のステップ
基本的な生存から効率的なリソース収集へと移行するには、以下の特定の進行パスに従ってください：

<Steps>
  <Step num="1" title="スティックフェーズ">
    地面から棒を拾うか、小さな茂みを壊して集めます。**棒**と一般的なブロック（土/砂）を組み合わせて、最初の**木製のつるはし**を作成します。
  </Step>
  <Step num="2" title="骨へのアップグレード">
    小さなモブを狩るか、遺骸を見つけて**骨**を入手します。**骨**と**糸**（羊やドロップアイテムから入手）を組み合わせて、大幅に耐久性の高い**骨のつるはし**を作成します。
  </Step>
  <Step num="3" title="金属の統合">
    **アルミニウムブロック**を見つけたら、既存の木製のつるはしと融合させて標準的な**ピッケル**にアップグレードします。これにより、**磁鉄鉱**などのより硬い材料を採掘できるようになります。
  </Step>
</Steps>

### 魔法の杖と環境操作
杖は、**ボールワンド (Ball Wand)** のベースと、さまざまな生物学的または鉱物的なコンポーネントを組み合わせてクラフトされます。これらのアイテムは単にダメージを与えるだけでなく、周囲の世界のルールを変えてしまいます。

| 杖の名前 | レシピ | 効果 |
| :--- | :--- | :--- |
| **ボールワンド** | フレンドワンド + 糸 | 弱く基本的な飛び道具を発射 |
| **フレンドワンド** | ボールワンド + 分泌腺 | 小さな味方の召喚やバフ付与 |
| **ヘイトワンド** | ボールワンド + 生の羊肉 | 広範囲への大ダメージとブロック破壊 |
| **フェイスワンド** | フレンドワンド + 骨のつるはし | 予測不可能な環境変化 |

</Section>

<Section icon={Sparkles} title="移動に最適なアイテム TOP 5">

移動は、この **Lucid Blocks ガイド**の中で最も満足度の高い部分です。特定のアーティファクトを組み合わせることで、実質的にマップを飛び回り、落下ダメージを無効化し、到達不可能な高さにまで到達することができます。

**最高の機動性アイテム:**
- **ビーグローブ (Bee Glove)**: 絶対的な必須アイテム。滑空を可能にし、正しく使用すれば落下ダメージを無効化します。
- **グラップラー (Grappler)**: ビーグローブと組み合わせると、プレイヤーを驚異的な速度で射出します。
- **ヘイズハンド (Haze Hand)**: ダメージはゼロですが、敵（または自分自身）を激しく空中に打ち上げることができます。
- **リスの指輪 (Squirrel Ring)**: ジャンプ力が向上し、垂直方向のアスレチックに不可欠です。
- **黄金のブーツ (Golden Boots)**: 体力を +4 増やしますが、重量の物理法則を劇的に変化させ、独自の移動テクニックを可能にします。

<Callout type="info" title="鉛筆 (The Pencil)">
**鉛筆**を見逃さないでください。武器ではありませんが、**バブルベア**に名前を付けることができ、個人のクオリアを管理し、仲間を整理するために不可欠です。
</Callout>

### 究極の機動性セットのクラフト

| アイテム | 材料 | ボーナス |

| **羽 (Feather)** | 羊毛 + 棒 + 砂 | ベースコンポーネント |
| **ビーグローブ** | 羽 + 羊毛 | 滑空 / 落下無効 |
| **メタルフック** | アルミニウム + 糸 | グラップリングコンポーネント |
| **バグリング** | 磁鉄鉱 + フリオニド | 移動速度 +1 |

</Section>
