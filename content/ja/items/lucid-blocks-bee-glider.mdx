Lucid Blocksの広大な世界において、移動能力の差は進行の遅さと効率的な探索の差に直結します。ゲーム内で技術的に「ビーグローブ」と呼ばれる**Lucid Blocks ビーグライダー**は、プレイヤーが優先的に手に入れるべき最初の主要な移動アップグレードです。巨大な山を登る時も、浮島の間の危険な隙間を飛び越えようとする時も、このツールは滞空を維持するために必要な揚力を提供してくれます。多くの新規プレイヤーはレシピに必要な特定のコンポーネントを見つけるのに苦労し、希少な紐や複雑な機械を探してプロセスを難しく考えがちです。実際には、どの環境オブジェクトに干渉すべきかを知っていれば、飛行への道は驚くほどシンプルです。この2026年版ガイドでは、冒険の早い段階でLucid Blocksの空をマスターできるよう、正確なクラフト要件、材料の場所、および高度な滑空テクニックを詳しく解説します。

<Section icon={Box} title="ビーグローブ・グライダーのクラフト">

ビーグローブは、地上での移動と空中での自由の架け橋となる序盤の傑作アイテムです。専用のクラフトステーションを必要とする終盤のアイテムとは異なり、探し方さえ知っていればビーグローブは比較的すぐに手に入れることができます。

| 材料 | 数量 | 主な入手先 | レアリティ |
| :--- | :--- | :--- | :--- |
| **羽** | 1 | 段ボール箱 / 環境ドロップ | アンコモン |
| **ウール** | 1 | コンテナ / 資源収集 | コモン |

<Callout type="tip" title="レシピ解放のヒント">
ビーグローブを解放するために設計図は必要ありません。インベントリに**羽**と**ウール**を同時に持っているだけで、クラフトメニューにレシピが表示されます。
</Callout>

多くのプレイヤーは、グライダーの製作に紐やシルクが必要だと誤解しています。しかし、2026年のメタでは、上記の2つの有機材料のみが必要であることが確認されています。これにより、**Lucid Blocks ビーグライダー**はゲーム内で最もコストパフォーマンスの高いツールの1つとなっており、非常に少ないリソース投資で絶大な利便性を提供します。

| 特徴 | ビーグローブ（グライダー） | 通常のジャンプ |
| :--- | :--- | :--- |
| **滞空時間** | 長い（滑空） | 最小限 |
| **水平移動距離** | 優秀 | 非常に短い |
| **落下ダメージ** | 無効（起動中） | 高いリスク |
| **必要条件** | 羽 + ウール | なし |

</Section>

<Section icon={Map} title="クラフト材料の入手場所">

**Lucid Blocks ビーグライダー**の材料を見つけるには、環境への干渉に対する鋭い目が必要です。ウールは豊富にありますが、羽は徹底的に探索しないプレイヤーにとっては見つけにくい場合があります。

**動画のハイライト:**
- 都市部と農村部での段ボール箱のスポーン地点の特定。
- ウールを入手するための効率的なコンテナ探索。
- 一般的なドロップ品とレアなドロップ品の見た目の違い。

<Callout type="warning" title="探索戦略">
羽はすべてのコンテナから確定でドロップするわけではありません。早く見つける確率を最大化するには、積極的に**段ボール箱を壊す**必要があります。
</Callout>

### 羽の探し方
羽はビーグローブにおける「ボトルネック」となるリソースです。主にマップ中に散らばっている段ボール箱の中から見つかります。これらの箱は破壊可能なオブジェクトで、多くの場合、様々なクラフトコンポーネントが隠されています。
- **場所**: 放棄された建造物、キャンプサイトの近く、建物の中などで見つかります。
- **方法**: 任意のツールや武器を使って箱を壊すと、戦利品が自動的にドロップします。
- **プロのヒント**: 箱をただ通り過ぎないでください。すべての箱が羽の潜在的な供給源です。

### ウールの集め方
ウールは入手が大幅に容易で、通常の探索中に収集されることが多いです。
- **場所**: 一般的な戦利品コンテナ、チェスト、および草原バイオームでの資源収集の副産物として見つかります。
- **インベントリ管理**: 羽がドロップした瞬間に準備ができているよう、常に少なくとも1つのウールを保管庫に置いておきましょう。

| 場所のタイプ | 期待できるドロップ | 干渉方法 |
| :--- | :--- | :--- |
| **放棄されたキャンプ** | ウール、スクラップ | コンテナを調べる |
| **段ボール箱** | **羽**、紙 | オブジェクトを壊す |
| **補給チェスト** | ウール、ツール | チェストを開ける |

</Section>

<Section icon={Wind} title="飛行の技術をマスターする">

Lucid Blocksでの飛行は、翼を羽ばたかせることではなく、慣性と重力を管理することです。**Lucid Blocks ビーグライダー**は、降下速度を落とし、垂直方向の落下を水平方向の速度に変換する、パラシュートとグライダーのハイブリッドとして機能します。

<CardGrid cols={3}>
  <Card title="開始地点の高さ">
    開始地点が高ければ高いほど、より遠くまで移動できます。ジャンプする前に、常に山や高い塔を探しましょう。
  </Card>
  <Card title="起動のタイミング">
    最大の前進慣性を維持するために、ジャンプの頂点に達した直後にビーグローブを起動してください。
  </Card>
  <Card title="慣性制御">
    移動キーを前に押し続けてください。キーを離すと失速し、垂直方向に早く落下してしまいます。
  </Card>
</CardGrid>

<Callout type="success" title="高度な戦術">
「浮島」からビーグローブを使用すると、障害物に当たらない限り、一度の滑空でマップの約50%を横断することができます。
</Callout>

効果的に飛ぶために、以下の環境的なヒントに従ってください：
1. **山**: 長距離移動のための最高の自然な発射地点です。
2. **高い建造物**: プレイヤーが作った塔は、恒久的な「飛行ハブ」として機能します。
3. **高い崖**: 戦闘からの脱出や峡谷を渡るのに理想的です。

| 飛行指標 | 低いジャンプ（地上） | 高いジャンプ（崖） | 極端なジャンプ（山） |
| :--- | :--- | :--- | :--- |
| **滑空時間** | 2〜5秒 | 15〜30秒 | 60秒以上 |
| **移動距離** | 最小限 | 中程度 | 絶大 |
| **用途** | 戦闘回避 | 隙間の横断 | マップ横断 |

</Section>

<Section icon={Target} title="ステップバイステップ・クラフトガイド">

空へ飛び立つ準備ができたら、以下の手順に従って記録的な速さで**Lucid Blocks ビーグライダー**を確保しましょう。

<Steps>
  <Step num="1" title="ウールを探す">
    開始エリアを探索し、見つけたすべてのコンテナやチェストを開けます。ウールは一般的なドロップ品で、プレイ開始から5〜10分以内に見つかるはずです。
  </Step>
  <Step num="2" title="段ボール箱を狩る">
    建造物の近くにある段ボール箱を探します。基本的なツールを使ってそれらを壊してください。羽がドロップするまでこれを繰り返します。
  </Step>
  <Step num="3" title="クラフトメニューを開く">
    両方のアイテムがインベントリに入ったら、クラフトインターフェースを開きます。ビーグローブが「ツール」または「装備」タブに表示されるようになります。
  </Step>
  <Step num="4" title="装備して発射">
    ビーグローブをアクティブな装備スロットに配置します。高い段差を見つけ、ジャンプし、空中でもう一度ジャンプキー（通常は起動キー）を押します。
  </Step>
</Steps>

<Callout type="info" title="インベントリのヒント">
レシピが表示されない場合は、アイテムが近くの収納箱ではなく、実際の*キャラクター*のインベントリにあることを再確認してください。
</Callout>

</Section>
