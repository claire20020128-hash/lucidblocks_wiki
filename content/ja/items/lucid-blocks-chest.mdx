Lucid Blocksの開発者は、従来のクラフトメニューをApotheosisシステムに置き換えました。リストからレシピをクリックする代わりに、最大6つのアイテムを手動でグリッドに配置する必要があります。ゲームはこれらの「供物」のエッセンスを計算して結果を決定します。この実験こそが、新しい**Lucid Blocks チェスト**のバリエーションや高度なツールを発見する唯一の方法です。

<CardGrid cols={3}>
  <Card title="エッセンスの論理">
    - **類似エッセンス**: 同種のアイテム（例：金属）を混ぜると、予測可能な高ティアのツールが得られます。
    - **ランダムな混合**: 無関係なアイテムを混ぜると、「奇妙なアーティファクト」が生成されることがあります。
    - **スロットの使用**: 基本的なレシピに6スロットすべてを使う必要はありません。アイテムが少ないほど安定性が増します。
  </Card>
  <Card title="希釈のコツ">
    - **安価な充填材**: 空きスロットに土や砂を加えて、基本ツールのレシピを埋めます。
    - **数量アップ**: 希釈により、最終製品の受け取り量が増加します。
    - **ティア制限**: このトリックは、高ティアのアーティファクトや武器には機能しません。
  </Card>
  <Card title="変異の確率">
    - **複雑性**: 6スロットすべてを埋めると、「奇妙な変異」が発生する確率が高まります。
    - **安定性**: 確実なストレージブロックの結果を得るには、2〜3個のコアコンポーネントに絞ってください。
    - **失敗**: 融合に失敗した場合は、正しいバランスを見つけるためにアイテムを1つずつ入れ替えてみましょう。
  </Card>
</CardGrid>

<Section icon={Map} title="必須の Lucid Blocks チェストの選択肢">

採掘を開始した瞬間からストレージが最優先事項となります。**Lucid Blocks チェスト**ユニットにはいくつかのタイプがあり、それぞれ拠点の経済において異なる目的を果たします。キャッシュキューブが標準的ですが、キャビネットやファンネルなどの他のオプションも特定のレイアウトで独自の利点を提供します。

### ストレージブロックの比較

| ブロック名 | 主要材料 | ストレージタイプ | 機能 |
| :--- | :--- | :--- | :--- |
| **キャッシュキューブ** | 磁鉄鉱、プリズムキューブ | チェストコンテナ | 大容量、汎用ストレージ |
| **キャビネット** | 糸、カットウッドブロック | 小型コンテナ | コンパクト、序盤用 |
| **ファンネル** | 磁鉄鉱、カットウッドブロック | ホッパーブロック | 自動アイテム輸送 |
| **アクセロブロック** | キラ、グー | ユーティリティ | プレイヤーやアイテムの垂直射出 |

<Callout type="warning" title="インベントリ管理">
インベントリがいっぱいになるまで最初の**Lucid Blocks チェスト**作成を待たないでください。序盤の探索では多くの「一点物」のアイテムが手に入り、スロットを圧迫します。保管室の準備ができていないと、貴重な材料を捨てざるを得なくなります。
</Callout>

### 代替ストレージソリューション
プリズムキューブが不足している場合は、クラフトチェーンを利用して大量のストレージを生成できます。磁鉄鉱1つとカットウッドブロック1つを組み合わせることで、ファンネルを作成できます。その後、ファンネル1つと別のカットウッドブロックを融合させると、3つのキャビネットが生成されます。これは序盤に大規模なストレージエリアを構築するための最もリソース効率の良い方法です。

</Section>

<Section icon={Target} title="最初のキャッシュキューブの作り方">

キャッシュキューブは、中盤の進行において最も信頼できる**Lucid Blocks チェスト**です。クリーンなUIと十分なスロットを提供し、大量のリソース管理を可能にします。以下の手順に従って、最初の高ティアストレージユニットを確保しましょう。

<Steps>
  <Step num="1" title="磁鉄鉱を集める">
    岩の多いバイオームや地下の洞窟で磁鉄鉱を見つけます。キャッシュキューブやファンネルを1つ作るごとに、少なくとも1つの破片が必要になります。
  </Step>
  <Step num="2" title="プリズムキューブを入手する">
    プリズムキューブは、標高の高いバイオームで見つかるか、マナジェムの地形ブロックを融合させることで入手できます。これはキャッシュキューブの「安定剤」として機能します。
  </Step>
  <Step num="3" title="Apotheosis UIにアクセスする">
    クラフトインターフェースを開き、スロットに磁鉄鉱1つとプリズムキューブ1つを配置します。変異を避けるため、他の「エッセンス」アイテムは追加しないでください。
  </Step>
  <Step num="4" title="融合を完了させる">
    クラフトプロセスを開始します。1つのキャッシュキューブストレージブロックを受け取り、拠点の好きな場所に設置してアイテムの保管を開始できます。
  </Step>
</Steps>

<Callout type="success" title="大量生産">
経験豊富なプレイヤーは、高地のバイオームでマナジェムの地形ブロックをファームすることが多いです。これらのブロックは磁鉄鉱とユニークな相互作用を持ち、材料さえあれば1回のクラフトセッションで数十個のキャッシュキューブを生成できます。
</Callout>

</Section>

<Section icon={Swords} title="採取ツールと基本武器">

材料を採掘するためのツールがなければ、**Lucid Blocks チェスト**を構築することはできません。Apotheosisシステムは、武器や採取ギアも管理しています。以下は、大規模なストレージ施設を建設する前に確保しておくべき必須レシピです。

### 必須の採取ツール

| ツール名 | 材料 | 用途 |
| :--- | :--- | :--- |
| **木のツルハシ** | 棒、土/丸太/砂 | 基本的な開始ツール |
| **葉のツルハシ** | 棒、木のツルハシ | 葉やベリーの高速採取 |
| **骨のツルハシ** | 骨、糸 | 石材用の中レベルの耐久性 |
| **霜のツルハシ** | 棒、オスミウム/フリオニド | 寒冷バイオームでの高速採掘 |
| **タングステン付属肢**| 水銀の触手、アルミニウム | 重要な中盤コンポーネント |

### 保護のための戦闘装備

磁鉄鉱を採掘している間、敵対的なエンティティに遭遇することがあります。リソースノードを守ることは、採掘そのものと同じくらい重要です。

  <Card title="近接武器">

    - **触手のレイピア**: アズラエル + 磁鉄鉱（高速で高精度のアップグレード）。
    - **アメジストナイフ**: メイス + 羊毛（近接戦闘での高い殺傷力）。
    - **硫黄の剣**: 硫黄 + タングステン付属肢（敵を炎上させる）。
  </Card>
  <Card title="魔法の杖">
    - **ボールワンド**: フレンドワンド + 糸（基本的な飛び道具）。
    - **ヘイトワンド**: ボールワンド + 生の羊肉（破壊的な範囲ダメージ）。
    - **フェイスワンド**: フレンドワンド + 骨のツルハシ（予測不能な環境効果）。

  </Card>
</CardGrid>
