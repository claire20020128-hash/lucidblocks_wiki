最も重要な隠しメカニズムの一つが**希釈（Dilution）**です。基本的なアイテムの場合、6つのスロットすべてを高価値の材料で埋める必要はありません。1つか2つの核となる材料を使い、残りのスロットを土や砂などの「充填材」で埋めることで、生産されるアイテムの数を大幅に増やすことができます。ただし、注意してください。上位のアーティファクトや専用ツールでこれを試みると、多くの場合合成に失敗するか、「役に立たないゴミ（Useless Junk）」アイテムになってしまいます。

</Callout>

<Section icon={Hammer} title="序盤の進行に欠かせないツールレシピ">

Lucid Blocks の序盤数時間における主な目標は、資源の獲得です。木製ピッケルのような標準的なツールは簡単にクラフトできますが、耐久性と速度に欠けます。世界のより深い層へと進むには、金属ベースまたは属性ベースのツールにアップグレードする必要があります。以下に挙げる**Lucid Blocks クラフトレシピ**は、序盤のハードルをクリアするための最も効率的な選択肢です。

**フロストピッケル**（またはアイスピッケル）は、序盤で最高の投資先であると広く考えられています。木製や骨製よりも採掘速度が大幅に速く、オスミウムの堆積場所を見つけることができれば比較的安価に作成できます。有機素材の収集に重点を置くプレイヤーにとって、**リーフピッケル**は壊れやすいものの、葉から棒やベリーを即座に収穫できる特化型ツールです。

| ツール名 | 材料 | 主な機能 |
| --- | --- | --- |
| **木製ピッケル** | 棒 + 土 / 丸太 / 砂 | 基本的な採掘とブロック破壊 |
| **リーフピッケル** | 棒 + 木製ピッケル | 葉やベリーの即時収穫 |
| **骨のピッケル** | 骨 + 紐 | 石の採掘用の中層耐久度 |
| **フロストピッケル** | 棒 + オスミウム / フリオナイド | 序盤の鉱石用の高速採掘 |
| **プラムアックス** | 木製ピッケル + 磁鉄鉱 | 高度な伐採と大規模な整地 |
| **つるはし** | 木製ピッケル + アルミニウム (x3) | 標準的な重作業用採掘ツール |

<Callout type="tip" title="持続可能性のヒント">
常に在庫に棒と紐をストックしておきましょう。ほとんどのツールは専用の作業台がなくてもその場でクラフトできるため、拠点に戻ることなく長時間フィールドに留まることができます。
</Callout>

<Section icon={Zap} title="移動装備とサバイバルアーティファクト">

探索はこの体験の核心ですが、地形はしばしば垂直で危険です。移動用装備のクラフトはオプションではなく、レイラインや高高度の構造物に到達するための必須条件です。**ビーグローブ**（グライダーグローブとも呼ばれる）は、空中で滑空し落下ダメージを無効化できるため、序盤にクラフトできる最も重要なユーティリティアイテムです。

**メタルフック**や**グラップラー**と組み合わせることで、ビーグローブは「スリングショット」のような移動を可能にします。フックで自分を打ち出し、その後にグローブを展開することで、通常では横断不可能な広大な距離を移動できます。

| 装備名 | 材料 | 恩恵 |
| --- | --- | --- |
| **ビーグローブ** | 羽 + 羊毛 | 滑空による落下ダメージの無効化 |
| **メタルフック** | アルミニウム + 紐 | 垂直方向の登攀と移動 |
| **ヘイズハンド** | 羊毛 + 小石ブロック (x2) | 敵を吹き飛ばす（ダメージゼロ） |
| **バグリング** | 磁鉄鉱 + フリオナイド | パッシブ 移動速度 +1 |
| **リスの指輪** | バグリング + 切り出した木材 | パッシブ ジャンプ力 +1 |
| **ゴールデンブーツ** | 漏れやすいペンダント + 水銀 | 重力物理を変更、体力 +4 |

<Steps>
  <Step num="1" title="羊毛と羽を集める">
    平原にいる羊のような生物を見つけ、羊毛を収穫します。羽は巣の中や特定の木を伐採することで見つかります。
  </Step>
  <Step num="2" title="ビーグローブをクラフトする">
    アポテオシスグリッドで羽と羊毛を組み合わせ、滑空能力をアンロックします。
  </Step>
  <Step num="3" title="アルミニウムを入手する">
    岩だらけのバイオームでアルミニウム鉱床を採掘し、垂直移動用のメタルフックをクラフトします。
  </Step>
  <Step num="4" title="移動をシナジーさせる">
    フックを使用して高い場所に到達し、ジャンプしてから滑空アクションを押し続けることで、マップを安全に移動します。
  </Step>
</Steps>

</Callout>

<Section icon={Shield} title="拠点のユーティリティと自動化">

最終的には、増え続ける材料コレクションを保管する場所が必要になります。ユーティリティアイテムの**Lucid Blocks クラフトレシピ**は、保管、自動化、および拠点の防衛に焦点を当てています。**キャビネット**は標準的な保管ソリューションですが、上級プレイヤーはアイテムの自動仕分けのために**漏斗（ファンネル）**の導入を検討すべきです。

多層階の拠点を構築する場合、**アクセロブロック**は不可欠なコンポーネントです。このブロックを踏むと、プレイヤーやアイテムが上方に射出され、原始的ですが効果的なエレベーターシステムとして機能します。

| ユーティリティアイテム | 材料 | 用途 |
| --- | --- | --- |
| **キャビネット** | 紐 + 切り出した木材ブロック | 標準的なアイテム保管 |
| **漏斗（ファンネル）** | 磁鉄鉱 + 切り出した木材ブロック | 自動ホッパーと仕分け |
| **アクセロブロック** | キラ + グー | プレイヤーやアイテムの垂直射出 |
| **ライトピンサー** | グラッグルトークン + 硫黄の剣 + フック | 高強度の拠点照明 |
