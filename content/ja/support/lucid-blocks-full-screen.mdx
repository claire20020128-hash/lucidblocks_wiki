**lucid blocks フルスクリーン** を有効にするには、ゲームプレイ中にいつでも **F11** を押すだけです。シェーダー効果や特定のバイオームで使用されている「フラクタル」数学を十分に堪能するために、この設定を強くお勧めします。

| アクション | キー / 入力 | 説明 |
| :--- | :--- | :--- |
| **フルスクリーン** | **F11** | ウィンドウモードとフルスクリーンモードを切り替えます |
| **移動** | **WASD** | 標準的な方向移動 |
| **ジャンプ** | **Space** | 垂直移動とクライミング |
| **しゃがむ** | **Left Ctrl** | 縁からの落下を防止（Divide の探索に必須） |
| **インベントリ** | **E** | メインメニューと Apotheosis タブを開きます |
| **ブロックを壊す** | **左クリック** | 標準的なインタラクション/攻撃 |
| **ブロックを置く** | **右クリック** | サブのインタラクション/建築 |
| **アイテムを落とす** | **Q** | アクティブスロットからアイテムを捨てます |

<Callout type="tip" title="プロのヒント：しゃがむことによる安全性">
「Divide（境界）」――地上と下の世界を隔てる分厚い天井――を探索するときは、常に **Left Ctrl** を押し続けてください。グラップリングフックやグライダーを装備していない限り、Divide からの落下はほぼ確実に致命的となります。
</Callout>

<Section icon={Sparkles} title="Apotheosis 錬金術システム">

Lucid Blocks におけるクラフトは **Apotheosis**（神格化/変成）と呼ばれます。これは従来の組み立てというよりも、変成や錬金術のように機能します。物質はある状態から別の状態へと転送され、しばしば予測不能でシュールな結果をもたらします。単に道具を「作る」のではなく、集めた材料からそれを「呼び起こす」のです。

<Steps>
  <Step num="1" title="Apotheosis タブにアクセスする">
    **E** キーを押してインベントリを開き、画面上のボタンまたはキーボードの矢印キーを使用して **Apotheosis** タブに移動します。
  </Step>
  <Step num="2" title="材料を選択する">
    円形の錬金術インターフェース内のいずれかのスロットに、2 種類の異なる材料を置きます。特定のスロットは重要ではなく、アイテムの種類と量のみが影響します。
  </Step>
  <Step num="3" title="経験値を獲得する">
    変成に成功するたびに、目に見えない XP を獲得します。メーターが一杯になると「Bloom（開花）」し、ステータスをアップグレードできるようになります。
  </Step>
  <Step num="4" title="結果を受け取る">
    中央に新しいアイテムが表示されます。一部のレシピは互換性があることに注意してください。例えば、グラップリングフックは Mana Gel（マナジェル）または Mercury Tendrils（水銀の触手）で作ることができます。
  </Step>
</Steps>

| 完成アイテム | 材料 A | 材料 B | 効果 / 用途 |
| :--- | :--- | :--- | :--- |
| **木のツルハシ** | 木のブロック | 棒 | 基本的な採掘ツール |
| **グラップリングフック** | マナジェル | 糸 | 垂直移動に不可欠 |
| **アヒルのブレスレット** | 砂 | 羊毛 | 空中制御+5、HP+2（装備可能） |
| **名前の書き換え鉛筆** | 本 | 糸 | アイテムやエンティティに名前を付ける |
| **水バケツ** | 空のバケツ | 木のブロック | 木を液体の水に変成させる |
| **パドル** | ツルハシ | 生の羊肉 | 奇妙な「刺す」タイプの近接武器 |

<Callout type="warning" title="死亡時のインベントリ変成">
Lucid Blocks における死は、変容のイベントです。インベントリ内のすべてのものがランダムに「神格化（変成）」されます。元のアイテムは失われますが、以前の装備から作られた、全く新しいレアな材料を持ってリスポーンすることもあります。
</Callout>
