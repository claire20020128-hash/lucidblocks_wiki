<Section icon={Sparkles} title="Apotheosis クラフトシステム">

Lucid Blocks におけるクラフトは Apotheosis（神格化）として知られています。従来のクラフトグリッドとは異なり、このシステムは「詠唱」とエッセンスの組み合わせに依存しています。**Lucid Blocks マルチプレイヤー**のワールド構築をマスターするには、コアとなるクオリアと組み合わせたときに様々な素材がどのように反応するかを理解する必要があります。

  <Card title="エッセンスの組み合わせ">

    - **憎悪のエッセンス**: ダメージは増加するが、リスクも危険なほど上昇する高リスク素材。

    - **リーフピック**: 森林をテーマにしたクオリアで効率を発揮する再生ツール。

<Section icon={Shield} title="サバイバルと拠点建築">

リミナルな広がりの中で生き残るには、単に家を建てるだけでは不十分です。魂を世界に繋ぎ止める必要があります。世界は手続き型で生成され、無限に感じられるため、リスポーン地点を設定することが最優先事項となります。

<Steps>
  <Step num="1" title="再生アンカーを入手する">
    生命（Life）と信仰（Faith）のステータスをレベルアップして、再生アンカー（Rejuvenation Anchor）ブロックを受け取ります。これが永続的な転生位置を設定する唯一の方法です。
  </Step>
  <Step num="2" title="資源の供給源を確保する">
    **オパールベリー**や**豆腐**が豊富なクオリアインスタンスを見つけましょう。食料は乏しく、特定の消耗品なしでは体力は自動的に回復しません。
  </Step>
  <Step num="3" title="シェルターを建設する">
    **壁紙ブロック**や**土ブロック**を使用して、中をくり抜いたスペースを作ります。「スクエア・バリアント」のようなエンティティは、開けた場所で見つかると非常に凶暴です。
  </Step>
  <Step num="4" title="クオリアリンクを確立する">
    個人のクオリアプラントに戻るポータルを作成します。これにより、大空のどれほど深くを探索していても、安全な避難所を確保できます。
  </Step>
</Steps>

<Callout type="success" title="食料の効率">
**魚の死骸**は、即座に得られる体力回復量が最も高いです。水中盆地や氷の平原を探索する際は、常にインベントリにいくつか忍ばせておきましょう。
</Callout>

| 食料アイテム | 体力回復量 | レア度 |
| :--- | :--- | :--- |
| **オパールベリー** | ハート 1 個 | 一般 |
| **豆腐の塊** | ハート 2 個 | アンコモン |
| **魚の死骸** | ハート 4 個 | レア |
| **バブルベアの卵** | ハート 3 個 + バフ | 極めてレア |
| **アイスキャンディー** | ハート 1.5 個 | 箱の中から発見 |

</Section>

<Section icon={Activity} title="エンティティと戦闘戦術">

Lucid Blocks のエンティティは、しばしば「難解な生き物（esoteric critters）」と呼ばれます。無害なものもいれば、執拗なものもいます。**Lucid Blocks マルチプレイヤー**の議論において、プレイヤーは「メタルプレイヤー（Metal Player）」や「スクエア・バリアント」への対処法について戦略を共有することがよくあります。

**戦闘戦略チェックリスト:**
- **距離を保つ**: ほとんどの敵のリーチは短いです。「引き打ち（walk-back）」テクニックを使用しましょう。
- **頭を狙う**: マネキンのようなエンティティは、頭上からの攻撃で受けるダメージが増加します。
- **環境物理を利用する**: 多くの敵は水中に誘い込むことができ、沈んで溺死させることが可能です。
- **爆弾の使用**: 水銀爆弾は、メタルプレイヤーの集団や殺害不可能な巨大クリーチャーのために取っておきましょう。

| エンティティ | 行動 | 主なドロップ |
| :--- | :--- | :--- |
| **スクエア・バリアント** | 攻撃的、素早い飛びかかり | ボールエッセンス、粘土 |
| **メタルプレイヤー** | 高 HP、金属音 | アルミニウム、オスミウム |
| **バブルベア** | 攻撃されない限り受動的 | バブルグー |
| **マネキン** | 遠くから忍び寄る | 棒、木のピック |
| **ジャイアント** | 不死身、ブロックを破壊 | なし（絶対に避けること） |

<Callout type="info" title="巨大なクリーチャー">
最近のコミュニティの報告によると、ブロックを破壊してトンネルを作る巨大なクリーチャーが目撃されています。現在は殺害不可能です。見かけたらすぐに**リーキーペンダント**を使用して出口を見つけてください。
</Callout>

</Section>
