<Section icon={Map} title="移動のマスターと移動装備">

Lucid Blocks の垂直な地形では、歩行は非効率的です。移動関連の実績を解除するには、飛行や急速なグラップリングを可能にする特殊な装備をクラフトする必要があります。マップの各地に隠された **Lucid Blocks ティアマナ・レイライン** に到達するには、グライダーグローブとフックショットの組み合わせをマスターすることが唯一の道です。

| アイテム | レシピ | 実績 | 使用例 |
| :--- | :--- | :--- | :--- |
| **グライダーグローブ** | 羊毛 + 羽 (骨 + 羊毛) | **Virtue α** | 高い山頂から滑空して距離を稼ぐ |
| **フックショット** | メタルブロック + 紐 | **Virtue β** | 表面に引っ掛けて勢いを維持する |
| **羽** | 骨 + 羊毛 | なし | グライダーグローブの構成部品 |

<Callout type="info" title="移動コンボ">
フックショットの勢いとグライダーグローブを組み合わせることができます。高い場所にグラップリングして、スイングの頂点でグライダーを展開することで、巨大な溝を飛び越えることができます。
</Callout>

</Section>

<Section icon={Swords} title="魔法の兵器：杖のクラフト">

Lucid Blocks での戦闘は「Archangel」実績を中心に展開され、特定の魔法武器を装備する必要があります。マネキンのような敵からドロップすることもありますが、ボス戦に耐えうる耐久力を確保するために、自分でクラフトすることをお勧めします。

| 杖のタイプ | レシピチェーン | 実績 |
| :--- | :--- | :--- |
| **Ball Wand** | 磁鉄鉱 + 棒 = Key Axe; Key Axe + 磁鉄鉱 | **Archangel α** |
| **Faith Wand** | Bubble Wand + 骨 = Friend Wand; Friend Wand + 骨 | **Archangel β** |
| **Explosive** | Accelo Block + Ball Essence (または 火 + フック + Bubble Wand) | **Archangel γ** |

<Callout type="warning" title="武器の耐久度">
敵がドロップした杖は耐久度が非常に低いことが多いです。重要な戦闘中に壊れないよう、常に最初から自分でクラフトする方が安全です。
</Callout>

</Section>

<Section icon={Target} title="エコシステムの操作とファーミング">

「Dominion」と「Apotheosis β」の実績を解除するには、現地の動植物に対して神のように振る舞う必要があります。これには、無限のリソースを提供する生命体を作り出すための特定の融合チェーンが含まれます。

<Steps>
  <Step num="1" title="草ブロックの作成">
    **土** と **砂** を混ぜて草ブロックを作成します。これはほとんどの有機融合の基礎となります。
  </Step>
  <Step num="2" title="生物パーツの採取">
    現地の生物を倒してセグメントや腺を採取します。これを **草ブロック** と組み合わせて **Bubblebear Egg** を生成します。
  </Step>
  <Step num="3" title="Fruit Femme の生成">
    **Bubblebear Egg** と別の **草ブロック** を融合して **Fruit Femme Capsule** を作成します。この生物は成長すると無限に食料を提供します。
  </Step>
  <Step num="4" title="受粉と収穫">
    Fruit Femme を倒して種を入手します。それを植え、肥料を使用して **Dominion α** を解除し、パールが育つのを待って **Dominion β** を解除します。
  </Step>
</Steps>

<Callout type="success" title="無限食料ループ">
Fruit Femme Capsule はゲーム内で最も価値のあるアイテムの一つです。空腹回復アイテムが枯渇しないよう、クオリアの中で大切に保管してください。
</Callout>

</Section>

<Section icon={Shield} title="ティアマナ・レイラインと最終ボス">

エンドゲームは、**Lucid Blocks ティアマナ・レイライン** の探索から始まります。これらの白く光るテレポートブロックは、ロザリオを完成させ最終ボスを召喚するための唯一の手段です。すべての **Lucid Blocks ティアマナ・レイライン** を見つけるには、十分なレベリングと適切なアップグレードが必要です。

| 実績 | 必要条件 | 戦略 |
| :--- | :--- | :--- |
| **Angel γ** | ティアマナ・レイラインを発見する | 野生にある白く光るブロックを探す |
| **Angel δ** | ティアマナの試練を乗り越える | 白いキューブに入り、内部の試練をクリアする |
| **Apotheosis γ** | 神聖なアーティファクトを融合する | 草ブロック + Ball Essence (Kodama の精霊から入手) を混ぜる |
| **Angel ε** | 最終ボスに到達する | すべてのレイラインをクリアしてロザリオを完成させる |

<Callout type="info" title="レイラインの追跡">
**Lucid Blocks ティアマナ・レイライン** を見つける最も速い方法は、レベルアップ時に **Pendant Upgrade** を選択することです。このアップグレードにより、マップ上に近くのレイラインの位置が表示されます。
</Callout>

これらのアップグレードに必要な経験値を稼ぐには、**無限融合ループ** を使用してください：
1. **Goo Capsule** と任意のアイテムを融合して **Gel Capsule** を入手します。
2. **Gel Capsule** とアイテムを融合して **2つの Goo Capsule** を入手します。
3. このプロセスを繰り返して無限に生物を生成し、大量の XP を獲得します。

</Section>
